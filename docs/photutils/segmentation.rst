Segmentation Photometry and Properties
======================================

.. warning::
    `scikit-image`_ is required for some functionality.

.. _scikit-image: http://scikit-image.org/


`photutils.segmentation` provides functions to calculate photometry
and morphological properties of sources from a segmentation map.  If
working with segmentation images, The
:func:`~photutils.segmentation.segment_properties` function calculates
the properties for all (or a specified subset) of segmented sources.
Please see `~photutils.segmentation.SegmentProperties` for the list of
properties that are calculated.
